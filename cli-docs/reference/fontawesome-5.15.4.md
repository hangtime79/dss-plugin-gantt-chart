# FontAwesome 5.15.4 Reference

Font Awesome 5.15.4 organizes icons into styles. The most common for plugins are **Solid** (`fas`) and **Brands** (`fab`).

## Common Icons

### General Interface
| Icon Class | Description |
|------------|-------------|
| `fas fa-puzzle-piece` | Generic plugin icon |
| `fas fa-cogs` | Settings or processing |
| `fas fa-wrench` | Tools or configuration |
| `fas fa-home` | Home or dashboard |
| `fas fa-info-circle` | Information |
| `fas fa-check-circle` | Success or completion |
| `fas fa-exclamation-triangle` | Warning or alert |
| `fas fa-times` | Close or remove |
| `fas fa-search` | Search |
| `fas fa-filter` | Filter |

## Migration Note
FontAwesome 5 uses a different syntax than version 3.2.1:
- **Old:** `icon-name` (e.g., `icon-cog`)
- **New:** `fas fa-name` (e.g., `fas fa-cog`)
