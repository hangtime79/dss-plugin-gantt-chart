## Technical Discoveries

### Platform Behavior
- **Webapp icons require inline SVG**: FontAwesome classes (`fas fa-*`) don't work in Dataiku webapp context. Must use inline SVG with `fill="currentColor"` for theme compatibility.

### Library Behavior
- None new this release

### Architecture Insights
- **Getting Started as overlay**: Using a full-page overlay with `display: none` toggle is cleaner than trying to show/hide multiple elements. The gs-* prefixed classes keep it isolated.

## Lessons Learned

1. **Scope expansion can be positive**: The Getting Started Guide is more valuable than the original loading state spec
2. **Platform limitations surface during implementation**: Always test visual elements in actual Dataiku context early
3. **Overlay hiding needs to be comprehensive**: When showing an overlay, hide ALL underlying content, not just some of it
