# Release Notes: v0.4.1.5

**Release Date:** 2025-12-22
**Type:** Documentation
**Branch:** `docs/v0.4.1.5-github-issues-config`

---

## Summary

This release focuses on improving the developer experience by upgrading GitHub Issue templates to structured YAML forms and restructuring internal protocol documentation.

---

## Changes

### Added
- **GitHub Issue Forms:** Replaced legacy Markdown templates with interactive YAML forms for Bug Reports and Feature Requests.
- **Issue Management Protocol:** Added `cli-docs/cli-protocols/issue-management-protocol.md` to standardize issue tracking.

### Changed
- **Protocol Structure:** Moved all branch protocols from `plan/` to `cli-docs/cli-protocols/` for better organization.
- **CLAUDE.md:** Updated references to point to the new protocol locations.
