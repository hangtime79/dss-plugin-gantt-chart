# Release Notes: v0.7.2

**Release Date:** 2025-12-27
**Type:** Bugfix
**Branch:** `bugfix/v0.7.2-markers-and-zoom`

---

## Summary

Fixes expected progress marker alignment at all view granularities and improves zoom behavior with mandatory stops at common percentages.

---

## Changes

### Fixed
- **Expected progress markers now align perfectly** (#52)
  - Markers use the Today line's X position directly
  - Guarantees alignment at Hour, Day, Week, Month, Year views
  - Simpler and more reliable than time-based calculation

### Changed
- **Zoom now passes through mandatory stops** (#53)
  - Stops at 25%, 50%, 75%, 100%, 150%, 200%
  - Uses 5px increments between stops
  - No maximum zoom limit - can zoom in indefinitely
  - View floor still enforced for zoom out
