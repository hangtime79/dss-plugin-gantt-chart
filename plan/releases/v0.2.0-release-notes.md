# Release v0.2.0

## Features
- **Custom Tooltips:** Added ability for users to select specific columns to display in the task details popup.
  - New `Tooltip Fields` configuration parameter in the chart settings.
  - Supports multiple columns with drag-and-drop reordering.
  - Handles various data types (dates, numbers, text) and null values gracefully.

## Bug Fixes
- **Popup Date Display:** Fixed issue where task dates were displayed as "Undefined to Undefined" or "N/A to N/A" in the popup.
  - Resolved issue with Frappe Gantt passing a wrapper object (`{task: ..., chart: ...}`) instead of the task object directly.
  - Improved date formatting logic to handle both string and Date objects.
- **Cyclic Object Error:** Fixed `TypeError: cyclic object value` when logging task objects for debugging.

## Technical Details
- Updated `webapp.json` to include `tooltipColumns` of type `DATASET_COLUMNS`.
- Modified `TaskTransformer` to extract and structure custom fields.
- Updated frontend `app.js` to render custom fields in the popup.
- Added CSS styling for custom tooltip fields.
- Added unit tests for `TaskTransformer` custom field logic.
