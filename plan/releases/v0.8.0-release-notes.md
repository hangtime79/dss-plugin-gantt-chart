# Release Notes: v0.8.0

**Release Date:** 2025-12-27
**Type:** Feature
**Branch:** `feature/v0.8.0-headers-and-date-formats`

---

## Summary

Headers & Date Formats milestone delivers a refined timeline header experience with reduced height, visual column separators, consistent year context across all view modes, decade grouping in Year view, and configurable date formats for task popups.

---

## Changes

### Added
- **Vertical header separators** (#50): Subtle vertical dividers between header columns for clearer visual demarcation
  - 1px lines centered with lower-text elements
  - Light gray color (#dfe6e9) for minimal visual noise
  - Auto-update on view mode and zoom changes

- **Configurable date format** (#35): New "Date Format" dropdown in configuration
  - ISO 8601: `2024-12-27`
  - US: `12/27/2024`
  - European: `27/12/2024`
  - Long: `December 27, 2024`
  - Short: `Dec 27`
  - Applied to all popup date displays

- **Decade grouping in Year view** (#14): Upper headers show decade labels
  - Format: "2020s", "2030s", etc.
  - Individual years remain in lower headers
  - No duplicate decade labels within same decade span

### Changed
- **Header height reduced by ~30%** (#41):
  - Control bar: 56px → 44px
  - Grid header: 50px → 38px
  - All text remains readable

- **Year context in upper headers** (#12): Year now visible in Day/Week/Month views
  - Full format at wide widths: "December 2025"
  - Abbreviated at narrow widths: "Dec 2025"
  - Month view shows years in upper row (existing)
  - Year view shows decades in upper row (new)
