# Release Notes: v0.4.1.6

**Release Date:** 2025-12-22
**Type:** Documentation
**Branch:** `docs/v0.4.1.6-protocol-updates`

---

## Summary

This release integrates GitHub Issues into the development workflow and refines the branch protocols. It ensures that specifications, branches, and pull requests are explicitly linked to tracked issues, and automates milestone management.

---

## Changes

### Changed
- **Branch Open Protocol:** Updated to require triage of GitHub Issues and linking of specifications to issues. Restored critical "What You Must NOT Do" and "Common AI Mistakes" sections.
- **Branch Exit Protocol:** Updated PR creation process to automatically include "Fixes #issue" keywords for auto-closing.
- **Branch Post-Merge Protocol:** Added automated step to close GitHub Milestones after release.
- **Protocol Index:** Reflected the new location of protocol files.
